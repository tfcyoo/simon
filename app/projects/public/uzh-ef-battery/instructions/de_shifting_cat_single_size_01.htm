Anleitung (2 von 4)

Wie schnell sind Sie?

Ihnen werden verschiedene Bilder präsentiert, die Sie anhand ihrer Grösse (kleiner/grösser als ein Fussball) kategorisieren sollen.

Ein visueller Hinweis soll Sie an die Aufgabenstellung erinnern:



Drücken Sie die linke Pfeiltaste, wenn das abgebildete Tier oder Objekt kleiner als ein Fussball ist.

Drücken Sie die rechte Pfeiltaste, wenn das abgebildete Tier oder Objekt grösser als ein Fussball ist.

Bitte versuchen Sie so genau und so schnell wie möglich zu antworten.


Drücken Sie bitte die rechte Pfeiltaste, um zur nächsten Seite der Anleitung zu gelangen.
